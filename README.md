# BCT NGO Website Redesign Prototype

This is a GitHub Pages-ready static prototype for Bhoruka Charitable Trust.

## Files
- `index.html` — full one-page prototype
- `assets/css/style.css` — responsive UI styling
- `assets/js/main.js` — mobile menu, tabs, filters, scroll reveal

## How to publish on GitHub Pages
1. Create a new repository, for example: `bct-ngo-redesign`
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Under Build and deployment, choose Source: Deploy from a branch.
5. Select Branch: `main` and Folder: `/root`.
6. Save.
7. GitHub will generate a live URL like: `https://yourusername.github.io/bct-ngo-redesign/`
