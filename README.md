# BCT NGO Redesign Prototype

GitHub Pages-ready static prototype for Bhoruka Charitable Trust website redesign.

## What this version improves

- Complete header/footer page inventory included.
- All visible programme subpages are tracked.
- Donation UI redesigned with cause cards and trust block.
- Partner wall includes all visible current partner/funding agency names.
- Contact section preserves all visible office addresses, phone numbers and emails.
- Annual reports repository plan preserves visible reports from 2008-2009 to 2016-2017.
- UTF-8, responsive layout, SEO metadata and schema markup included.
- Custom SVG visuals included in `assets/img`.

## Publish on GitHub Pages

1. Upload the files/folders above to the repository root.
2. Go to Settings → Pages.
3. Source: Deploy from branch.
4. Branch: main, Folder: /root.
5. Save.

The existing live URL will update after GitHub Pages redeploys.
