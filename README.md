# BCT NGO Website Experience

GitHub Pages-ready static website layout for Bhoruka Charitable Trust.

## Included

- Modern responsive homepage.
- Header and footer aligned for desktop and mobile.
- Full page structure covering About, Programmes, Get Involved, Resources, Careers, Donate and Contact.
- Programme cards with category visuals.
- Donation section with cause cards and trust information.
- Partners, resources, annual reports and contact sections.
- SEO metadata, UTF-8 setup, semantic headings and schema markup.

## Publish

1. Upload the files and folders above to the repository root.
2. Go to Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main, Folder: /root.
5. Save and use the live GitHub Pages link.
